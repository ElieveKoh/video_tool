# 🎥 Video Tool v6.0

완전 자동 설치가 가능한 Streamlit 기반 비디오 변환 도구

## ✨ 주요 기능

- **🎯 스마트 코덱 변환**: H.264, H.265, VP9, AV1
- **📺 해상도 변환**: 4K ↔ HD ↔ SD 자유자재
- **🎞️ 프레임레이트 변환**: 23.976, 24, 25, 29.97, 30, 50, 59.94, 60 fps
- **📼 디인터레이싱**: Progressive / Interlaced 선택
- **🎛️ 커스텀 비트레이트**: 비디오/오디오 비트레이트 직접 설정
- **📥 유튜브 배치 다운로드**: 여러 영상을 Queue에 추가하여 일괄 다운로드/변환
- **🔄 실시간 모니터링**: 진행률, 예상 시간, 중단 기능
- **🆕 앱 자동 업데이트**: 실행 시 GitHub Releases 최신 버전 감지 후 원클릭 업데이트

## 🔄 자동 업데이트 (Mac)

- `실행하기.command` 또는 `VideoTool.app` 실행 시 최신 릴리즈를 자동 확인합니다.
- 새 버전이 있으면 `지금 업데이트할까요? [Y/n]` 안내가 표시됩니다.
- `Y`를 누르면 업데이트 파일 다운로드 후 자동 적용되고 앱이 재시작됩니다.
- 업데이트 소스: `https://github.com/ElieveKoh/video_tool/releases`

## 🚀 사용 방법

### 🍎 **Mac 사용자**
1. `실행하기.command` 파일을 더블클릭 (또는 `VideoTool.app` 실행)
2. 보안 경고가 뜨면: 시스템 설정 > 개인정보 보호 및 보안 > "확인 없이 열기" 클릭
3. 자동 설치 완료까지 대기 (최초 실행 시 3-5분 소요)
4. 브라우저에서 앱 자동 실행 (localhost:8601)

### 🪟 **Windows 사용자**
1. `start.bat` 파일을 더블클릭
2. 자동 설치 완료까지 대기 (최초 실행 시 3-5분 소요)
3. 브라우저에서 앱 자동 실행 (localhost:8601)
4. **주의**: Python 3.10 이상 필요 (설치 시 "Add Python to PATH" 체크 필수)

## 🛠 자동 설치 내용

- Python 3.11 (없는 경우)
- FFmpeg (비디오 처리 엔진)
- yt-dlp (유튜브 다운로드)
- Streamlit (웹 인터페이스)

모든 의존성은 로컬 `bin/` 폴더에 다운로드되어 시스템을 건드리지 않습니다.

## 💡 사용법

### **Tab 1: 로컬 파일 변환**
1. 📂 "Select Folder" 또는 "Select Files" 클릭
2. ✅ 변환할 파일들 체크박스 선택
3. 🎯 코덱, 해상도, 품질, 프레임레이트, 스캔 타입 선택
4. 💾 Custom Bitrate 선택 시 비디오/오디오 비트레이트 직접 입력
5. 🚀 "Start Conversion" 클릭
6. 📁 변환된 파일은 `converted_[코덱명]` 폴더에 저장

### **Tab 2: 유튜브 다운로드 + 변환**
1. 📥 "Download Only" 체크 여부 선택
   - ✅ 체크: 다운로드만 (변환 안 함)
   - ☐ 체크 해제: 다운로드 + 변환
2. 🎯 변환 설정 선택 (코덱, 해상도, 품질, 프레임레이트 등)
3. 📺 유튜브 URL 붙여넣기 → Enter
4. ✅ 비디오 정보 확인 후 자동으로 Queue에 추가
5. 🔁 다른 설정으로 다른 영상도 추가 가능
6. 📋 Download Queue에서 체크박스로 선택/해제
7. 🚀 "Start Batch Download/Convert" 클릭
8. 📁 다운로드/변환된 파일은 `youtube_downloads` 폴더에 저장

## ⚙️ 코덱별 특징

| 코덱 | 특징 | 권장 용도 |
|------|------|-----------|
| **H.264** | 범용 호환성 최고 | 일반적인 용도, 모든 기기 재생 |
| **H.265** | 50% 파일 크기 절약 | 저장공간 절약, 최신 기기 |
| **VP9** | 웹 최적화 | YouTube, 웹 스트리밍 |
| **AV1** | 차세대 최고 효율 | 미래형 고압축, 최신 브라우저 |

## 🎬 프레임레이트 가이드

- **23.976 fps**: 영화 필름 (Cinema)
- **24 fps**: 디지털 시네마
- **25 fps**: PAL (유럽/아시아)
- **29.97 fps**: NTSC (북미/일본 TV)
- **30 fps**: 디지털 비디오
- **50/59.94/60 fps**: 고프레임 (스포츠, 게임)

## 🎛️ 품질 프리셋

- **Fast**: 빠른 변환, 보통 품질
- **Balanced**: 균형잡힌 속도/품질 (권장)
- **High**: 느린 변환, 최고 품질
- **Custom Bitrate**: 비디오/오디오 비트레이트 직접 설정

## 📝 시스템 요구사항

- **Mac**: macOS 10.14 이상
- **여유 공간**: 원본 파일 크기의 1.5배 이상
- **인터넷**: 초기 설치 및 유튜브 다운로드 시 필요
- **메모리**: 최소 4GB RAM 권장

## 🆘 문제 해결

**보안 경고 (Mac):**
1. `실행하기.command` 더블클릭 시 "확인되지 않은 개발자" 경고
2. 시스템 설정 > 개인정보 보호 및 보안
3. "확인 없이 열기" 클릭

**설치 실패 시:**
- 인터넷 연결 확인
- 충분한 디스크 공간 확인 (최소 500MB)
- 터미널에서 직접 실행: `./실행하기.command`

**변환 실패 시:**
- 충분한 디스크 공간 확인
- 파일 권한 확인
- 다른 프로그램에서 파일 사용 중인지 확인
- FFmpeg 오류 메시지 확인

**유튜브 다운로드 실패 시:**
- URL이 올바른지 확인
- 비공개/제한 영상이 아닌지 확인
- yt-dlp가 최신 버전인지 확인 (자동 다운로드됨)

## 📂 폴더 구조

```
VideoTool/
├── 실행하기.command          # 실행 파일 (더블클릭)
├── video_converter_app.py    # 메인 앱
├── requirements.txt          # Python 패키지 목록
├── README.md                 # 사용 설명서
├── bin/                      # 자동 다운로드 바이너리 (FFmpeg, yt-dlp)
├── venv/                     # Python 가상환경 (자동 생성)
├── converted_[코덱명]/       # 변환된 로컬 파일 저장 폴더
└── youtube_downloads/        # 유튜브 다운로드 파일 저장 폴더
```
